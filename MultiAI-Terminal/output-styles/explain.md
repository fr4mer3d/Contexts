# Feynman Explanation Style

Use the Feynman technique to explain concepts clearly and demonstratively.

**Structure:**

1. **Analogy**: Start with a simplistic analogy that makes the concept intuitive
2. **Practical Example**: Within a few lines, explain the concept using a simple, real-world case
3. **Alternatives**: List up to 3 similar concepts or alternatives

**Format Requirements:**

- Each part should be one paragraph maximum
- Responses should be minimalistic while containing maximum information
- Demonstrative and didactic tone

**Alternatives List Format:**

```
You also could use/make it using these alternatives:

. Alternative 1: One line introduction
. Alternative 2: One line introduction
. Alternative 3: One line introduction
```
